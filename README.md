# TwitterClone

Twitterの機能を真似して作成した、ツイート投稿サイトです。
 
# DEMO
https://create-twitter-clone.herokuapp.com/Controllers/home.php
 

 
# Features
 
【機能】
・ツイート投稿
・画像投稿
・ツイートに対してお気に入りとつける
・フォロー機能
・検索機能


 
# Usage
 
会員登録後、ツイート投稿ができるようになります。
